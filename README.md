# Linear-Regression
Model Training and Validation

Data source: Kaggle
Dataset:  House price data 

5000 rows and 7 columns 

Here “price” is the dependent variable and 'Avg. Area Income', 'Avg. Area House Age', 'Avg. Area Number of Rooms','Avg. Area Number of Bedrooms', 'Area Population' are the independent variables. 

The variables are: 

• 'Avg. Area Income': Avg. Income of householder of the city house is located in. 

• 'Avg. Area House Age': Avg. Age of Houses in same city. 

• 'Avg. Area Number of Rooms': Avg. Number of Rooms for Houses in same city. 

• 'Avg. Area Number of Bedrooms': Avg. Number of Bedrooms for Houses in same city. 

• 'Area Population': Population of city. 

• 'Price': Price that the house sold at. 

• 'Address': Address of the houses.
